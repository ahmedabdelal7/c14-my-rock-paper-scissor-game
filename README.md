## 📌 Description
A Windows Forms game built with C# that simulates the classic Rock-Paper-Scissors game.

The player competes against the computer by choosing one of the three options, and the system determines the winner based on standard game rules. The game tracks rounds, results, and scores in real-time.

---

## ⚙️ Features
- Play against the computer
- Choose between Rock, Paper, and Scissors
- Random computer selection
- Round counter
- Score tracking (Player / Computer / Draw)
- Display game result after each round
- Reset rounds functionality
- Start screen and About Me screen

## 🏗️ How the System Works
1. The player starts the game from the start screen.
2. The user selects a weapon (Rock, Paper, or Scissors).
3. The computer randomly selects its choice.
4. The system compares both choices:
   - Determines win, lose, or draw
5. Scores are updated:
   - Player score
   - Computer score
   - Draw count
6. The result is displayed instantly.
7. The player can reset the game at any time.
